1- Aula sobre a estrutura de um HTML

Nesta aula, foi explicado a estrutura básica de um HTML, sobre o nome da html index sendo uma convesão a ser seguida, utilizando as tags básicas para a inicialização de uma pagina.

  • html: Tag inicial para toda página HTML
  • head: É a tag para linkar css e javascript e informações de matadados
  • title: Texto que aparece na topo do navegador/aba
  • body: O corpo da página HTML

2- Aula sobre TAGs

Nesta aula foi explicado algumas TAGs e seus usos, tags que não precisam de fechamentos e alguns tipos de atributos.

  • i: Transforma o texto entre as tags em Itálico
  • strong: Transforma o texto definido entre as tags em Negrito
  • input: Cria um campo para digitação (Entrada de dados)
  • img: link para direcionar o local da imagem
Voltar

3- Aula sobre Atributos

Nesta aula foi explicado alguns Atributos e algumas formas de seu uso e exemplificando na prática.

  • id: Determina um ID (identificador) para a Tag
  • style: Aplica um estilo a tag e para uso do CSS
  • class: Também utilizado para CSS
  • type: Atributo da tag Input, determina o tipo do campo
  • src: Atributo da tag Img, informa o URL ou URI da imagem

4- Aula sobre Textos

Nesta aula foi explicado algumas tags relacionadas a textos, explicando sobre titulos, parágrafos, citações e etc.

  • h1 a h6: Determina o tamanho dos títulos
  • p: Utiliza para fazer um parágrafo
  • blockquote: Utilizado para fazer citações
  • u: Aplica underline (sublinhado) a palavra
  • sup e sub: Sup é texto sobrescrito (acima) e sub é texto subscrito (abaixo)

5- Aula sobre Listas

Nesta aula foi explicado sobre listas, os tipos são ordenadas e não ordenadas.
Obs: uma lista pode ser inserida dentro de outra lista.

  • ol: Lista ordenada, utiliza números para ordenação
  • ul: Lista não ordenada, utiliza ponto para ordenar
  • li: É a tag filha de OL e UL para criar a lista

6- Aula sobre Links

Nesta aula foi explicado sobre links, os tipos de atributos e uso do target para manipular o modo de visualização no navegador.

  • href: É o endereço de URL ou URI da página desejada
  • target: É utilizado para escolher a ação a ser feita ao clicar no link
  • title: É para inserir um titulo ao link, passando o mouse em cima
